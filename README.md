# 表演評分系統

一個基於 React + Supabase 的即時表演評分系統，支援大螢幕控制台和觀眾評分介面。

## 功能特色

### 大螢幕控制台 (`/admin`)
- 密碼保護（預設密碼：8888）
- 輸入表演者名字，同步到所有觀眾端
- 動態滾動分數顯示（3位數+1位小數）
- 15秒倒數計時
- 即時顯示前8名平均分數
- 評分結束後顯示最終平均分數

### 觀眾評分介面 (`/`)
- 自動接收表演者名字
- 評分範圍：1-100（整數）
- 顯示個人評分
- 即時顯示前3名分數和平均分數
- 防止重複評分

## 安裝與執行

1. 安裝依賴：
```bash
npm install
```

2. 準備背景圖片：
將體育場背景圖片命名為 `stadium-background.jpg` 並放置在 `public` 資料夾中

3. 啟動開發伺服器：
```bash
npm run dev
```

4. 訪問：
- 觀眾評分：http://localhost:3000
- 大螢幕控制台：http://localhost:3000/admin

## Supabase 設定

系統已配置 Supabase 連接，請確保以下資料表已建立：

- `app_status` - 應用程式狀態表
- `ratings` - 評分紀錄表

詳細 SQL 請參考專案說明文件。
